# Projet site web : Milka web site

Ce projet individuel avait pour objectif principale d'appliquer les règles de base d'une convention CSS. Ici, c'est la convention de nommage BEM qui a été appliquée dans le cadre de cette exercice.
Le défi était ensuite de développer l'intégralité du site web pour une marque de notre choix. 
Les technologies utilisées sont :
- HTML/CSS
- Javascript
- Netlify
